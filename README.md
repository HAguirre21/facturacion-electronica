# 💳 Facturación Electrónica - Negocio Mayorista

Solución de software desarrollada íntegramente en **Python** utilizando la librería **Flet**. Este sistema está diseñado para optimizar los procesos de venta, control de inventarios y gestión comercial en entornos de alta demanda (mayoristas), ofreciendo una interfaz de usuario moderna y reactiva.

## 🚀 Características Principales

* **Interfaz con Flet:** UI interactiva y fluida que facilita la navegación del usuario.
* **Gestión de Ventas:** Procesamiento eficiente de facturas para negocios con alta rotación de productos.
* **Control de Inventarios:** Sistema preparado para el seguimiento de stock y movimientos de mercancía.
* **Persistencia de Datos:** Módulo dedicado para la administración de bases de datos integradas.
* **Instalación Simplificada:** Incluye recursos específicos para el despliegue y configuración inicial del sistema.
